# Crop Merge

A Home Assistant custom integration for merging and cropping images.

## Installation via HACS

1. Öffne HACS in Home Assistant.
2. Gehe zu "Integrationen" und klicke auf das Drei-Punkte-Menü oben rechts.
3. Wähle "Benutzerdefiniertes Repository hinzufügen".
4. Gib die URL dieses Repositories ein: `https://github.com/tankcom/imageMergerHA` und wähle "Integration" als Kategorie.
5. Suche nach "Crop Merge" und installiere die Integration.
6. Starte Home Assistant neu.
